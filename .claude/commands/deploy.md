# DEPLOY AGENT

You publish the site generated by the WEB agent to Vercel and save the resulting URL.

**Input:** `$ARGUMENTS` (format: `<web-folder-path> <business-name>`)

---

## STEP 0 — Verification

Check that `<path>/index.html` or `<path>/package.json` exists.

If it doesn't exist:
> "DEPLOY FAILED: site not found at <path>. Run /web <path to auditoria.json> first"

---

## STEP 1 — Vercel Configuration

If `vercel.json` does NOT exist, create it based on the detected stack:

**Vanilla HTML** (index.html exists without package.json):
```json
{ "version": 2, "builds": [{ "src": "**/*", "use": "@vercel/static" }] }
```

**Astro** (astro.config.mjs exists):
```json
{ "buildCommand": "npm run build", "outputDirectory": "dist", "framework": "astro" }
```

**React/Vite** (vite.config.js exists):
```json
{ "buildCommand": "npm run build", "outputDirectory": "dist", "framework": "vite" }
```

---

## STEP 2 — Deploy

**CRITICAL:** All commands must be executed INSIDE the site folder (`<path>`), not from the project root.

```bash
# FIRST: enter the site folder
cd <path>

# Install dependencies if package.json exists
npm install

# Deploy to production
vercel --prod --yes
```

**Post-deploy verification:** After obtaining the URL, confirm that CSS assets load correctly:
- If the site has `styles/` → the URL must serve `<url>/styles/design-system.css`
- If it's Astro/React → the build must have generated `dist/` with bundled CSS

Capture the production URL from the output (`Production: https://...`).
If it fails, report the full error without workarounds.

---

## STEP 3 — Save Result

Save to `deploy/outputs/<business-name>-deploy.json`:
```json
{
  "negocio": "",
  "url_produccion": "",
  "fecha_deploy": "",
  "stack": "html|astro|react",
  "carpeta_origen": ""
}
```

---

## STEP 4 — Confirm

Respond in a single line:
> "DEPLOY DONE: <url> → saved to deploy/outputs/<business-name>-deploy.json"

## Rules
- Do not modify site files during deploy
- If vercel CLI is not installed: "Install Vercel CLI with: npm i -g vercel"
- Zero additional explanation
